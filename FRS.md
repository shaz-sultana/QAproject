# FRS - Login Feature

FR-001: User can login with email and password

FR-002: Email must be in valid format
Example: user@email.com
Invalid: notanemail

FR-003: Password must be minimum 8 characters

FR-004: Wrong credentials show error message:
"Invalid email or password"

FR-005: After 5 wrong attempts account locks for 30 mins

FR-006: Successful login redirects to home page
